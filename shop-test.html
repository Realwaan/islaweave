Manual Testing Guide

  1. Search Test:
    • Open shop page
    • Type "guitar" in search box
    • Verify 2 guitar products appear
    • Clear search, verify all products return
  2. Filter Test:
    • Click "Baskets" filter button
    • Verify only basket products show
    • Click "All" to see all products again
  3. Add to Cart Test:
    • Click "Add to Cart" on any product
    • Verify cart drawer opens
    • Verify toast notification appears
    • Verify button shows "✓ Added" briefly
  4. Mobile Test:
    • Open DevTools (F12)
    • Toggle device toolbar (Ctrl+Shift+M)
    • Test on iPhone/Android view
    • Verify responsive layout works
  5. No Results Test:
    • Search for "zzzzz"
    • Verify "No products found" message appears

Common Issues & Solutions

Issue: Products not showing
Solution: Check browser console for errors. Verify cart.js is loaded before shop.js.
Issue: Add to cart not working
Solution: Verify addToCart() function exists in cart.js and showToast() is defined globally.
Issue: Search not filtering
Solution: Check that product-search input ID matches in shop.js.
Issue: Cart drawer not opening
Solution: Verify openCartDrawer() function is defined in cart.js.
